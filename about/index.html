👨‍💻 个人简介

📄 基本信息

这里是 Chlorine，可以叫我小氯，或是其他顺口一点的称呼也无妨。

目前旅居地球，就读于某挺不错的大学，专业是某还算不错的交叉学科。如果您好奇是哪个的话……嗯，反正不是霍格沃茨啦。不过我相信，相比于好奇我的背景，我在此处栽培的花草应当会更值得您花去一点闲暇的。

我对星座和 MBTI 之类带或者不带神秘色彩的人格分类持保留态度，也并不倾向于用一个或者几个词汇概括自己的意识形态。如果您希望快速地了解小氯是个怎样的人，读这篇 About 或许是个更好的选择。

📙 个人经历

其实也没什么经历可言。一颗小小哒原子，从世界的一个不起眼的角落诞生，走过若干的坎坷和坦途。目前正处在人类口中「最好的年纪」。

🪄 所爱与厌

小氯喜欢的东西有很多：喜欢可口的食物，喜欢甜甜的果茶，喜欢优雅的公式，喜欢精美的软件，喜欢隽永的书籍，喜欢好看的画作，喜欢高精尖的电子产品，喜欢在晴朗的下午在阳光灿烂的公共空间闲坐品茶，喜欢在夏天的晚上在晚风轻拂的操场和朋友们一起唱歌，喜欢世间的一切自由、美好与爱。

小氯的情绪大抵还算稳定，也没有什么特别的忌讳。但是小氯的确很难对一些事物有好感，例如 DDL、不洁净、海鲜、昆虫、内卷、容易断墨的笔、几十个 G 的安装包、傲慢、偏见和没有说明文档的代码。当然，若是面对谎言、背叛、贪婪、蔑视、处心积虑、恶意伤害之类，小氯不保证自己不会情绪失控，拿出自己的本体单质请您品尝。

🏠 园子简介

📜 基本信息

这里的完整名字是「Garden of Outlier」，没有中文名字。

这里是互联网中一个偏僻而籍籍无名的小岛，是我用日光，热茶和位流建造的秘密花园，是一片原教旨主义的知识旷野，也是一个灵魂的永久栖息地。

如果您觉得这个名字比较拗口，可以和小氯一样，叫这里「园子」。

📚 这里可能有什么

这里可能会随机刷新出各种乱七八糟的内容：从新的冒菜口味体验到后量子加密算法为何数学不可破，从坐飞机去看北极熊龇牙到用广义矩估计预测自己的考试成绩，从为什么你应该放弃 GitHub 到如何阻止搜索引擎乱看你的个人信息，从百年战争到底在打什么到瞪眼法求解散点图 Pearson 相关系数，从博客样式优化日志到作为助教我是怎么教学生写功利主义道德观的，从热牛奶的最佳温度探索到 Manifest der Kommunistischen Partei 读书笔记……

总而言之，咱搞的是通识教育嘛（俗称缝合怪），什么都会一点（骄傲脸）。

当然，上面的是一个比较完整的设想，小氯目前的知识尚且十不足一。

❌ 这里不会有什么

如上所述，小氯可能在这里讲任何自己感兴趣的事物，但有一件事绝对不会讲：政治问题。没别的意思——这个问题很容易引起不必要的争议，而这些争议对推进人类知识的进步帮助有限。我可以保证不越界，但如果您非要断章取义或者自行解读，恕小氯不负责。

🛠 技术架构

园子是一个静态小岛，其底层架构完全开源：

  • 框架：Hugo
  • 主题：Hermeneutics
  • 源代码托管：Codeberg 和 小氯自己的 Forgejo 实例
  • 评论系统：Artalk。

园子的部署和运行基于互联网大善人们提供的免费服务，这里请允许小氯对它们的慷慨致以真诚的谢意。

  • DNS：CloudFlare
  • 托管：Netlify
  • 备份托管：在 Vercel 和 Cloudflare Pages 进行了冷运行备份
  • 图床：Cloudflare R2
  • 静态资源加速：jsDelivr 及其镜像

📆 编年史

  • 过去的历史：园子大致建立于 2023 年末，期间历经了数次各种各样的变更，但由于最近的改版，这部分历史已经被隐去。您可以认为，下面这里就是当今园子的 Genesis。
  • 2024-04-19: 园子的重构初步完成，重新命名为「Garden of Outlier」。
  • 2025-04-21: 得益于持续的探索，园子彻底摆脱了对 GitHub 的依赖，可以使用 Codeberg + Netlify 进行全自动部署。部署平台切换至 Netlify。

园子的互动和内容一样，基本没有禁忌。唯一的规则是：

Sine delictis, sine politicis.

即是如此。

如果您真的想让小氯为难的话……那没办法，我可爱不代表我好欺负。小氯有信心让您的评论在被任何第三个人看到之前就去 ~/.Trash 养老，也不保证尊重您的被遗忘权。

此外，如果您希望自己的发言是理性而令人愉快的，推荐 Občanské fórum 的八条规则。当然，这不是硬性规定。

ℹ️ 隐私政策

📈 统计分析

园子目前暂未添加任何统计或分析服务。我倒是想添加一个，但不知道用什么。

🍪 Cookie 情况

园子是严格 Cookie-free 的，因此您可以对这方面完全放心。

💬 评论隐私

评论的数据托管在小氯的个人 VPS 上，位于垃圾王 RackNerd 的 LA 机房。虽说服务只要和牢美挂边隐私性就很可能值得怀疑，但小氯也没找到更好的方案。小氯想过注册一个 Hetzner，可惜没成功。

评论数据会保留（要不然那不叫评论），日志的话，除非系统整个炸掉需要 debug，不然小氯没时间看。

数据分享的话，只要您不在评论区搞事，小氯用自己的呆毛发誓不会把您的邮箱之类的信息交给任何第三方。唯一的风险是服务商强行窥探（极小概率，因为这严重违反了服务商的隐私协议），不过小氯的大部分个人项目也托管在上面，出了这种事咱们一起寄（不是）。

前言

静态博客工作流程：


    sequenceDiagram
      participant picgo
      participant typora
      typora->>hexo: 生成md文件
      loop 选择一个博客主题
          hexo->hexo: 生成静态博客网页
      end
      participant github
      participant cloudflare
      hexo->>github: 通过git推送至gitHub仓库<br/>部署至github pages<br/>可绑定自己的域名
      cloudflare->>github: 静态网页CDN加速
     Note right of github: 网页端查看 <br/>博客...
     picgo-->>github: 1->github作为图床存储图片
     github-->>picgo: 你礼貌么？
     cloudflare-->>github: 2->使用Pages构建和部署无服务器进行图片CDN加速
     picgo->>typora: 3->提供图片链接
     Note right of typora: 调整 <br/>博客内容

Typora

一款 Markdown 编辑器和阅读器。

风格极简 / 多种主题 / 支持 macOS，Windows 及 Linux 实时预览 / 图片与文字 / 代码块 / 数学公式 / 图表 目录大纲 / 文件管理 / 导入与导出 ……

PicGo

一个用于快速上传图片并获取图片 URL 链接的工具。

灵活上传 自动复制链接 支持插件系统
主窗口支持拖拽、选择图片上传；支持读取剪贴板的第一张图片快捷上传。macOS支持顶部栏拖拽上传。 上传成功后自动复制链接地址到剪贴板，支持5种复制粘贴格式，让你的文本编辑极致高效。 支持JavaScript开发的插件，给予插件极大自由度，让PicGo更加强大，成为你得手的效率工具。

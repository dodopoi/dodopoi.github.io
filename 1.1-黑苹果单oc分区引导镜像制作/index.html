1.1 黑苹果单OC分区引导镜像制作

  • 2023-06-16
  • 2 分钟阅读
  • 828 字
  • 折腾
  • 黑苹果

写在前面：

需要在mac系统里进行，并在商店里下载好需要的镜像文件，这里以monterey12.6.7为例。

1、利用磁盘工具建立空白镜像，大小15GB，格式为扩展日志格式。

在终端里输入：

diskutil list

/dev/disk0 (internal, physical):

#: TYPE NAME SIZE IDENTIFIER

0: GUID_partition_scheme *300.1 GB disk0

1: EFI ⁨NO NAME⁩ 104.9 MB disk0s1

2: Microsoft Reserved ⁨⁩ 16.8 MB disk0s2

3: Microsoft Basic Data ⁨⁩ 192.0 GB disk0s3

4: Apple_APFS ⁨Container disk1⁩ 108.0 GB disk0s4

/dev/disk2 (disk image):

#: TYPE NAME SIZE IDENTIFIER

0: GUID_partition_scheme +15.0 GB disk2

1: EFI ⁨EFI⁩ 209.7 MB disk2s1

2: Apple_HFS ⁨mi⁩ 14.7 GB disk2s2

由上面可见建立的新的空白镜像文件disk2分为两部分：EFI分区和苹果镜像分区。将苹果镜像分区进行分区（一般包含PE分区和OC引导分区，即所谓双分区），这里只制作OC引导单分区。在终端里输入以下命令

sudo diskutil splitPartition disk2s2 JHFS+ macOS 14.7GB #单OC分区

sudo diskutil splitPartition disk2s2 2 MS-DOS PE 1g JHFS+ macOS 13.7GB #双分区OC+PE

sudo diskutil splitPartition disk2s2 JHFS+ macOS 14.7GB

2、将下载好的苹果镜像写入空白镜像

打开OpenCore Configurator挂载新建的EFI分区，将引导文件拷贝至EFI分区内(如果是双分区，把WinPE里的文件也拷贝至步骤1中创建的PE分区)。

下面是终端命令分解：

sudo

/Applications/Install\ macOS\ Monterey.app/Contents/Resources/createinstallmedia

#在访达里的应用程序里找到<安装macOS Monterey>右键·显示包内容，找到Contents/Resources/createinstallmedia，将createinstallmedia拖入到终端里。

–volume

/Volumes/macOS\ 1 /Applications/Install\ macOS\ Monterey.app

#将步骤1里在桌面生成的macOS和访达>应用程序里的<安装macOS Monterey>拖入到终端里。

–nointeraction

执行过程中需要输入你的开机账户密码。

sudo /Applications/Install\ macOS\ Monterey.app/Contents/Resources/createinstallmedia --volume /Volumes/macOS\ 1 /Applications/Install\ macOS\ Monterey.app --nointeraction
